# EE-559 : Deep Learning

Repository containing the two deep learning projects (one folder for each project).

[Project 1](https://github.com/armand33/deep_learning_epfl/tree/master/project1) : predictor of finger movements laterality from Electroencephalography (EEG) recordings.

[Project 2](https://github.com/armand33/deep_learning_epfl/tree/master/project2) : implementation from scratch of a DL framework capable of training a MLP with SGD.
